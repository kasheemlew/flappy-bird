# Flappy-Go

Flappy Bird Go Implementation. Inspired By Francesc Campoy

## Usage

### Download

```shell
$ go get https://github.com/kasheemlew/flappy-go
```

### Build

```shell
$ go build -i
```

### Run

```shell
$ ./flappy-go
```
